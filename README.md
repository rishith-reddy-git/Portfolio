# Soodini Rishith Reddy — Portfolio

A responsive, self-contained portfolio landing page. Open `index.html` in any modern browser to preview it.

## Personalize before publishing

- Replace `hello@rishithreddy.dev` in `index.html` with your email address.
- Replace the `#` social links with GitHub, LinkedIn, and Instagram URLs.
- Add your résumé and change the navigation button to point to it.
- Add real project links and gallery images as they become available.

## Deploy with GitHub Pages

1. Push the `master` branch to GitHub.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `master` branch and the `/(root)` folder, then save.
5. GitHub will publish the site at `https://rishith-reddy-git.github.io/Portfolio/`.

## Files

- `index.html` — page structure and content
- `styles.css` — responsive visual design
- `script.js` — navigation and interaction behavior

It can be deployed directly to GitHub Pages, Netlify, or Vercel without a build step.
